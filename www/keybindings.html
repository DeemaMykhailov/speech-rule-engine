Overview of key bindings

Key bindings for the Math Explorer

Key:
Shift SpaceActivate explorer when on maths expression
EscapeLeave exploration mode
 
Down ArrowExplore next lower level
Up ArrowExplore next upper level
Right ArrowExplore next element on level
Left ArrowExplore previous element on level
 
TabRepeat previous announcement
SpaceGet position information
EnterCollapse or expand expression under cursor, if possible
HomeGo to top level of expression
XSummary of expression under cursor
ZDetailed description of expression under cursor
VStart new virtual cursor
PGo to previous virtual cursor
UUndo all virtual cursors
 
Keys in Table:
 
Shift Down ArrowMove one cell down
Shift Up ArrowMove one cell up
Shift Right ArrowMove one cell right
Shift Left ArrowMove one cell left
 
0-9 0-9Move directly to cell (n,m) if it exists. (0,0) is cell (10,10).
